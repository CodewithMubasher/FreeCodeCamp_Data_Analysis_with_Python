## 🚀 Getting Started
 
### Prerequisites
 
Make sure you have Python 3.x installed along with the required libraries:
 
```bash
pip install pandas numpy matplotlib seaborn scipy
```
 
### Running a Project
 
Clone the repository and run any project file directly:
 
```bash
git clone https://github.com/CodewithMubasher/FreeCodeCamp_Data_Analysis_with_Python.git
cd FreeCodeCamp_Data_Analysis_with_Python
python mean_var_std.py
```
